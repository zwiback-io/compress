compress - a compact url shortener

Simple Website, API & Usage. Handles even the longest links like a proper chap.


Web Version: compress.zwiback.io/web

API

POST /new

Expects the to-be-shortened url in text format.
Returns the shortened url, no questions asked!

If the url is already present on the system, it will respond with the already existing shortened url.

GET /<id>

Retrieve the url with the given id and redirect to it.
If no url is found, return a 418 (it isn't a coffee-pot, after all).

DEL /<id>

Deletes the url with the given id.
